# Branding Guidelines - UTSA Registration Plus

These guidelines ensure the extension feels official, premium, and native to the UTSA ecosystem.

## 🎨 Color Palette

| Token             | Hex Code  | Usage                                                              |
| :---------------- | :-------- | :----------------------------------------------------------------- |
| **UTSA Orange**   | `#F15A22` | Primary actions (Save, Register), high-score badges, accent icons. |
| **Midnight Blue** | `#032044` | Headers, primary navigation, strong headings, primary buttons.     |
| **Limestone**     | `#F8F4F1` | Background for pages and secondary containers.                     |
| **Concrete**      | `#EBE6E2` | Borders, dividers, muted card backgrounds.                         |
| **White**         | `#FFFFFF` | Primary card surfaces, text on dark backgrounds.                   |

## Typography

- **Primary Font**: `Roboto` (Modern, high readability).
- **Alternative/Fallback**: `Arial` or sans-serif.
- **Heading Color**: Midnight Blue (`#032044`).
- **Action Color**: UTSA Orange (`#F15A22`).

## 🧩 UI/UX Standards

1.  **Card-Based Layout**: Group information (e.g., professor ratings, grade graphs) into distinct cards with `8px` corner radius.
2.  **Subtle Shadows**: Use soft shadows (`rgba(3, 32, 68, 0.1)`) instead of heavy borders.
3.  **Visual Hierarchy**: Use Midnight Blue for titles and UTSA Orange for interactive elements.
4.  **Accessibility**: Ensure a minimum 4.5:1 contrast ratio for all text.
